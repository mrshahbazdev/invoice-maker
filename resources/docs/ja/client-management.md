# クライアント管理と CRM: 詳細ガイド

顧客関係を効率的に管理することは、サービスまたは製品ベースのビジネスを成功させる上で重要です。すべての顧客データを整理して 1 か所に簡単にアクセスできるようにすることで、顧客とのコミュニケーション、請求、サポートを行う際に時間を欠かすことがなくなります。 

**クライアント** モジュールは、顧客関係管理 (CRM) ハブ全体として機能します。この 1 つのダッシュボードから、やり取りする個人や企業の生涯履歴全体を表示できます。

---

## 1. クライアント プロファイルの作成と管理
すべてのやり取りはクライアントのプロフィールから始まります。これは、顧客に関するすべての重要な情報が保存されるマスター レコードです。完全なプロファイルを作成すると、今後はワンクリックで請求書を発行できるようになります。

### 重要なビジネス情報
新しいクライアントを作成するときは、次のように入力できます。
- **連絡先の詳細:** 主要な電子メール アドレスと直通電話番号。ここで提供される電子メールは、見積書、請求書、支払い領収書を送信するために当社の自動メール システムによってのみ使用されます。
- **物理的な住所:** 完全な請求先住所 (番地、市区町村、州/県、郵便番号、および国) を保存できます。この情報は、作成した請求書の「請求先」セクションに自動的に取り込まれます。

### 税務コンプライアンスと法的分野
多くの法域では、B2B (企業間) の請求書発行には法的な正確さが求められます。
- **納税番号 / VAT ID:** 特定の納税者番号 (TIN) または VAT ID をクライアント プロファイルに直接保存できます。このクライアントの請求書を作成するたびに、その納税者 ID が最終的な PDF に自動的に埋め込まれ、手動でデータを入力しなくても、ビジネスが国内および国際の税法に準拠した状態を維持できるようになります。

## 2. インタラクティブなクライアント ダッシュボード
メイン リストからクライアントの名前をクリックすると、専用の 360 度インタラクティブなダッシュボードが開きます。この画面では、彼らとの経済的関係に関する即時の実用的な情報が提供されます。

### 生涯価値の指標
ダッシュボードの最上部には、次の内容がすぐに表示されます。
- **請求総額:** このクライアントにこれまでに送信されたすべての請求書の合計。
- **支払総額 (生涯価値):** あなたがそれらから集めた実際の収益。
- **未払い残高:** すべての未処理の請求書全体で現在支払われている正確な金額。これにより、クライアントから新しい仕事を受け入れる前に、クライアントの状態が良好かどうかをすぐに認識することができます。

### 文書履歴
財務指標の下では、文書全体の記録に即座にアクセスできます。 
- 請求書のグローバル リストをくまなく調べる必要はありません。このダッシュボードではすべてがフィルタリングされ、この 1 つのクライアントに固有の請求書と見積書のみが表示されます。
- どの請求書が起草、送信、一部、または全額支払われているかを確認できます。
- 送信されたすべての見積を表示して、どの見積が承認されたか拒否されたかを追跡することもできます。

## 3. 通信チャネルのカスタマイズ
すべてのビジネスの運営方法は異なります。形式的で高度に構造化された言葉遣いを好むクライアントもいますが、気軽でフレンドリーなチェックインに反応するクライアントもいます。別の人に請求するたびにメールを書き直す必要はありません。

### クライアント固有の電子メール テンプレート
デフォルトでは、アプリケーションはドキュメントを送信するときにグローバル電子メール テンプレート (ビジネス設定で構成) を利用します。ただし、クライアント管理システムを使用すると、これをクライアントごとに上書きできます。
- 特定のクライアントのプロファイル内の [**電子メール テンプレート**] サブタブに移動します。
- 完全にカスタムの件名行と電子メール本文を特別にカスタマイズして作成できます。
- このクライアント向けに新しい請求書または見積書を生成し、[電子メールで送信] をクリックすると、システムは一般設定を完全に無視し、パーソナライズされたテンプレートのみを使用します。 
- これにより、高度にパーソナライズされたタッチを大規模にシームレスに維持できるようになります。

## 4. 安全で対話型のクライアント ポータル
受信箱に埋もれてしまう静的で退屈な PDF 添付ファイルをクライアントに送信する代わりに、サブスクリプションには顧客専用のプレミアムでインタラクティブな Web アプリケーションが含まれています。

### パスワードレスの「Magic Link」認証
クライアントは、請求書を確認するためにアカウントを作成したり、別のパスワードを覚えたり、電子メールを確認したりする必要はありません。 
- 請求書を電子メールで送信すると、強力で暗号的に安全な「Magic Link」が含まれます。
- このリンクをクリックすると、即座に認証され、プライベート クライアント ポータルに直接ログインします。

### ステートメントビューと一括アクション
ポータルに入ると、クライアントは 1 つの請求書だけを見ているわけではありません。彼らはあなたとの関係全体を見ています。
- **明細書の概要:** 未払い総額を示す美しい概要が表示されます。
- **オンライン支払い:** ポータルは Stripe と直接統合されています。クライアントは未払いの請求書を表示し、「今すぐ支払う」をクリックして、クレジット カードを使用して残高を即座に決済できます。システムは、トランザクションが完了したミリ秒後に、内部請求書を自動的に「支払い済み」としてマークします。
- **一括ダウンロード:** クライアントは、会計上の目的で、過去のすべての請求書のコピーを要求することがよくあります。クライアントは、個別に電子メールで送信する代わりに、ポータルにログインし、過去の請求書の 10 件の横にあるチェックボックスをオンにして、[選択したものをダウンロード] をクリックすることができます。システムは、すべての PDF を含むきちんとした ZIP ファイルを同時に生成します。

### 統合サポートチケット発行
乱雑な電子メール スレッドでサポート リクエストが失われるのを防ぎます。
- クライアント ポータル内で、顧客は [新しいチケットを開く] をクリックして質問することができます (例: 「請求先住所を更新できますか?」または「最近の請求書の項目 #3 について質問があります」)。
- このチケットは管理ダッシュボードに即座に表示されます。 
- インターフェースから直接返信できます。クライアントは電子メール通知を受け取り、ポータルに直接応答できるため、会話全体がスレッド化され、CRM プロファイルに永続的にリンクされたままになります。
