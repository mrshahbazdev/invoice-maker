# Spesenverfolgung und -verwaltung: Detaillierte Anleitung

Bei einem profitablen Unternehmen kommt es nicht nur darauf an, wie viel Umsatz Sie erwirtschaften; Es geht darum, sorgfältig zu überwachen, wie viel Geld ausgegeben wird. Das Modul **Ausgaben** fungiert als Ihr digitaler Aktenschrank und stellt sicher, dass Sie nie eine Quittung verlieren und immer genau wissen, wofür Ihr Betriebskapital ausgegeben wird.

---

## 1. Eine neue Ausgabe protokollieren
Wenn Sie Software oder Ausrüstung kaufen oder geschäftlich reisen, sollten Sie dies sofort protokollieren. Navigieren Sie zur Registerkarte **Ausgaben** und klicken Sie auf **Ausgaben protokollieren**.

### Kategorisierung und Organisation
Die richtige Kategorisierung ist der Schlüssel zu umsetzbaren Finanzdaten am Jahresende.
- **Ausgabenkategorie:** Wählen Sie aus, um welche Art von Ausgabe es sich handelte (z. B. „Software-Abonnement“, „Bürobedarf“, „Reisen“, „Mahlzeiten und Unterhaltung“). Dadurch wird der Abfluss kategorisiert, sodass Ihr Kassenbuch genaue Kreisdiagramme Ihrer Ausgabegewohnheiten erstellen kann.
- **Betrag und Datum:** Protokollieren Sie den genauen gezahlten Betrag und das Datum, an dem die Transaktion stattgefunden hat.
- **Anbieter/Händler:** Notieren Sie genau, wen Sie bezahlt haben (z. B. „Amazon“, „Delta Airlines“, „Microsoft“). Dies hilft Ihnen, wiederkehrende Kosten im Laufe der Zeit zu erkennen.

---

## 2. Quittungs- und Dokumentenspeicherung
Wenn Sie geprüft werden, reicht ein Einzelposten in einer Tabelle nicht aus; Sie benötigen die tatsächliche Quittung. Die Plattform bietet sichere, cloudbasierte Dokumentenanhänge.

- **Upload-Feld:** Wenn Sie eine Ausgabe protokollieren, können Sie auf „Datei anhängen“ klicken und den PDF-Beleg, das JPEG-Foto der Restaurantrechnung oder einen Screenshot der digitalen Transaktion hochladen.
- **Permanente Speicherung:** Diese Datei wird sicher verschlüsselt und dauerhaft an die Spesenaufzeichnung in der Cloud angehängt. Sie können Ihren physischen Schuhkarton voller Quittungen buchstäblich wegwerfen – Ihr gesamter Prüfpfad ist digitalisiert und durchsuchbar.

---

## 3. Abrechnungsfähige Ausgaben (Neuabrechnung an den Kunden)
Oftmals entstehen Ihnen im Namen eines bestimmten Kunden Kosten, zu deren Erstattung dieser Ihnen vertraglich verpflichtet ist (z. B. Server-Hosting-Gebühren, Fotografiekäufe oder Reisetickets).

### Der Re-Billing-Workflow
Die Plattform meistert dieses komplexe Buchhaltungsmanöver einwandfrei:
1. Achten Sie beim Protokollieren der Ausgabe (z. B. ein Foto im Wert von 50 $) auf das Dropdown-Menü „Dem Kunden zuweisen“**.
2. Wählen Sie den spezifischen Kunden aus, für den das Produkt gekauft wurde.
3. Aktivieren Sie das Kontrollkästchen „Abrechenbar“**.
4. Das System markiert diesen Aufwand intern. 
5. Wenn Sie das nächste Mal eine neue Rechnung für diesen bestimmten Kunden erstellen, werden Sie vom System darüber informiert, dass für diesen Kunden noch nicht in Rechnung gestellte Ausgaben anfallen. Mit einem Klick können Sie die Fotokosten in Höhe von 50 US-Dollar direkt als Einzelposten auf die Rechnung übertragen und so sicherstellen, dass Sie nie vergessen, die Auslagen erstattet zu bekommen.

---

## 4. Globale Finanzintegration
Das Ausgabenmodul existiert nicht im luftleeren Raum. Es ist fest im **Kassenbuch** und im Buchhaltungskern verankert.

- Jede einzelne Ausgabe, die Sie protokollieren, wird sofort von Ihrer Nettogewinnkennzahl im Haupt-Dashboard abgezogen.
- Wenn Sie eine Ausgabe falsch kategorisieren, können Sie sie jederzeit bearbeiten, und die Kassenbuchberechnung berechnet Ihre Gewinn- und Verlustrechnungen (Gewinn und Verlust) in Echtzeit auf der gesamten Plattform neu.
