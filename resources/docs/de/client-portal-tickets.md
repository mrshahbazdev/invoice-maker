# Das Kundenportal und Support-System: Vollständiger Überblick

Den Kunden ein erstklassiges, erstklassiges Erlebnis zu bieten, unterscheidet moderne Unternehmen von veralteten Mitbewerbern. Anstatt statische PDFs per E-Mail zu versenden, die oft in Spamfiltern hängen bleiben, erhalten Sie mit Ihrem Abonnement eine sichere, interaktive Webanwendung, die ausschließlich für Ihre Kunden entwickelt wurde.

---

## 1. Zugriff auf das Kundenportal
Sicherheit und Benutzerfreundlichkeit gehen selten Hand in Hand, aber das Kundenportal nutzt einen modernen Authentifizierungsstandard, um Reibungsverluste zu vermeiden.

### Passwortlose „Magic Link“-Authentifizierung
Ihre Kunden müssen auf dieser Plattform kein Konto erstellen. Sie müssen sich kein komplexes Passwort merken und kein Profil verwalten.
- Wenn Sie eine Rechnung oder einen Kostenvoranschlag über Ihr Verwaltungs-Dashboard versenden, erhält der Empfänger eine professionell formatierte E-Mail.
- In dieser E-Mail befindet sich ein großer, sicherer Button, der einen kryptografisch gehashten „Magic Link“ enthält.
- Wenn der Kunde auf die Schaltfläche klickt, überprüft das System das Token in Millisekunden und meldet ihn automatisch direkt über seinen Browser in seinem dedizierten Portal an, wodurch 100 % der Anmeldeprobleme entfallen.

---

## 2. Rechnungen und Abrechnungsverwaltung
Sobald Ihr Kunde das Portal betritt, sieht er sich nicht nur die einzelne Rechnung an, die Sie ihm gerade gesendet haben. Sie betrachten die gesamte finanzielle Vergangenheit Ihres Unternehmens.

### Die interaktive Erklärung
- **Lebenslange Gesamtsummen:** Sie können sofort den Gesamtbetrag sehen, den sie Ihnen in der Vergangenheit gezahlt haben, zusammen mit dem genauen Dollarbetrag, den sie derzeit für alle ausstehenden Rechnungen schulden.
- **Rechnungsauflistung:** Sie können eine Tabelle anzeigen, in der jede Rechnung aufgeführt ist, die jemals an sie gesendet wurde. Sie können diese Liste dynamisch nach Status (Entwurf, Gesendet, Bezahlt, Überfällig) oder nach Ausstellungsdatum sortieren.
- **Dynamische Vorschauen:** Durch Klicken auf eine Rechnung wird diese sicher im Browser geöffnet und perfekt formatiert, unabhängig davon, ob sie sich auf einem riesigen Desktop-Monitor oder einem kleinen Mobiltelefon befinden.

### Massen-Download
Buchhalter lieben das Kundenportal wegen dieser Funktion.
- Während der Steuersaison bitten Ihre Kunden Sie möglicherweise, jede Rechnung aus dem Vorjahr erneut zu senden. Sie müssen dies nicht manuell tun.
- Raten Sie ihnen, auf ihren Magic Link zu klicken, das Kontrollkästchen „Alles auswählen“ auf ihrem Dashboard zu aktivieren und auf **„Alle herunterladen“ zu klicken.
- Das System packt alle historischen PDFs automatisch in einen einzigen, sauberen ZIP-Ordner, der direkt auf den Computer heruntergeladen wird.

### Sofortige Online-Zahlungen
Wenn Sie Stripe in Ihren Unternehmenseinstellungen konfiguriert haben, können Ihre Kunden Vorgänge sofort abwickeln.
- Auf einer ausstehenden Rechnung wird eine auffällige Schaltfläche **„Jetzt bezahlen“** angezeigt.
– Wenn Sie darauf klicken, wird ein sicheres, integriertes Stripe-Checkout-Fenster direkt im Portal geöffnet.
- Sie geben ihre Kreditkarteninformationen sicher ein. Nach der Verarbeitung wird das Portal sofort aktualisiert und Ihr eigenes Verwaltungs-Dashboard markiert die Rechnung automatisch als bezahlt. Es ist kein manueller Abgleich erforderlich.

---

## 3. Das interaktive Support-System
Die Kommunikation im Zusammenhang mit Abrechnungsvorgängen kann unglaublich chaotisch werden, wenn sie in E-Mail-Threads oder Textnachrichten versinkt. Das Kundenportal verfügt über ein integriertes Support-Ticket-System zur Zentralisierung und Nachverfolgung von Anfragen.

### Clientseitige Ticketerstellung
- Wenn ein Kunde eine Frage hat (z. B. „Können Sie meine Lieferadresse aktualisieren?“ oder „Ich habe eine Frage zu dieser Beratungsgebühr“), kann er in seinem Portal zur Registerkarte **Support** navigieren und auf **„Neues Ticket eröffnen“** klicken.
- Sie geben einen Betreff und eine Nachricht an und senden sie ab.
- Das Ticket wird sofort dauerhaft mit seinem CRM-Profil verknüpft.

### Geschäftsseitige Ticketlösung
- Zurück in Ihrem Verwaltungs-Dashboard wird auf der Registerkarte **Support-Tickets** ein Abzeichen angezeigt.
- Klicken Sie in das Ticket, um die Nachricht unseres Kunden zu lesen.
- Sie können direkt von Ihrer Benutzeroberfläche aus antworten. 
- *Die magische Aktion:* Unser System nimmt Ihre Antwort entgegen und sendet sie automatisch per E-Mail an den Kunden zurück. Der Kunde kann einfach auf diese E-Mail antworten und seine Antwort wird direkt zurück an den Ticket-Thread im System weitergeleitet.
- Dadurch wird sichergestellt, dass 100 % der Kommunikation zu diesem Kunden dauerhaft archiviert und direkt an sein Rechnungsprofil angehängt wird, wodurch verhindert wird, dass wichtige Informationen in isolierten Posteingängen verloren gehen.
