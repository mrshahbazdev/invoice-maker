# Kundenmanagement und CRM: Ausführlicher Leitfaden

Die effiziente Verwaltung Ihrer Kundenbeziehungen ist von zentraler Bedeutung für die Führung eines erfolgreichen service- oder produktbasierten Geschäfts. Wenn Sie alle Ihre Kundendaten an einem zentralen Ort organisiert und leicht zugänglich aufbewahren, verpassen Sie bei der Kommunikation, Abrechnung oder Betreuung Ihrer Kunden nichts. 

Das **Clients**-Modul fungiert als Ihr gesamter Customer Relationship Management (CRM)-Hub. Von diesem einzigen Dashboard aus können Sie den gesamten Lebensverlauf jeder Person oder jedes Unternehmens anzeigen, mit dem Sie interagieren.

---

## 1. Kundenprofile erstellen und verwalten
Jede Interaktion beginnt mit einem Kundenprofil. Dies ist der Stammdatensatz, in dem alle wichtigen Informationen zu einem Kunden gespeichert sind. Wenn Sie vollständige Profile haben, können Sie diese in Zukunft mit einem einzigen Klick in Rechnung stellen.

### Grundlegende Geschäftsinformationen
Beim Erstellen eines neuen Kunden können Sie Folgendes eingeben:
- **Kontaktdaten:** Ihre primäre E-Mail-Adresse und direkte Telefonnummer. Die hier angegebene E-Mail-Adresse wird von unserem automatisierten Mailingsystem ausschließlich zum Versenden von Kostenvoranschlägen, Rechnungen und Zahlungsbelegen verwendet.
- **Physische Adresse:** Sie können ihre vollständige Rechnungsadresse (Straße, Stadt, Bundesstaat/Provinz, Postleitzahl und Land) speichern. Diese Informationen werden automatisch in den Abschnitt „Rechnungsempfänger“ jeder Rechnung übernommen, die Sie für sie erstellen.

### Steuerkonformität und rechtliche Bereiche
In vielen Rechtsordnungen erfordert die Rechnungsstellung im B2B-Bereich (Business-to-Business) höchste rechtliche Präzision.
- **Steuernummer/Umsatzsteuer-ID:** Sie können eine bestimmte Steueridentifikationsnummer (TIN) oder Umsatzsteuer-ID direkt im Kundenprofil speichern. Wenn Sie für diesen Kunden eine Rechnung erstellen, wird dessen Steuernummer automatisch in das endgültige PDF eingebettet. So wird sichergestellt, dass Ihr Unternehmen ohne manuelle Dateneingabe die lokalen und internationalen Steuervorschriften einhält.

## 2. Das interaktive Client-Dashboard
Wenn Sie in der Hauptliste auf den Namen eines Kunden klicken, wird dessen spezielles interaktives 360-Grad-Dashboard geöffnet. Dieser Bildschirm liefert Ihnen sofortige, umsetzbare Informationen über Ihre finanzielle Beziehung zu ihnen.

### Lifetime-Value-Metriken
Ganz oben im Dashboard sehen Sie sofort:
- **In Rechnung gestellte Gesamtsumme:** Die Gesamtsumme aller Rechnungen, die jemals an diesen Kunden gesendet wurden.
- **Gezahlter Gesamtbetrag (Lifetime Value):** Der tatsächlich realisierte Umsatz, den Sie damit erzielt haben.
- **Ausstehender Saldo:** Der genaue Geldbetrag, den sie Ihnen derzeit über alle offenen Rechnungen schulden. Dies hilft Ihnen, sofort zu erkennen, ob ein Kunde einen guten Ruf hat, bevor Sie neue Aufträge von ihm annehmen.

### Dokumentverlauf
Unterhalb der Finanzkennzahlen haben Sie sofortigen Zugriff auf den gesamten Papierverlauf. 
- Sie müssen nicht Ihre globale Rechnungsliste durchsuchen. Dieses Dashboard filtert alles, um nur die Rechnungen und Kostenvoranschläge anzuzeigen, die für diesen einen Kunden spezifisch sind.
- Sie können sehen, welche Rechnungen erstellt, gesendet, teilweise oder vollständig bezahlt wurden.
- Sie können auch alle an sie gesendeten Kostenvoranschläge anzeigen und nachverfolgen, welche angenommen oder abgelehnt wurden.

## 3. Kommunikationskanäle anpassen
Jedes Unternehmen arbeitet anders. Manche Kunden bevorzugen eine formelle, stark strukturierte Sprache, während andere besser auf lockere, freundliche Check-ins reagieren. Sie sollten Ihre E-Mails nicht jedes Mal neu schreiben müssen, wenn Sie einer anderen Person eine Rechnung ausstellen.

### Kundenspezifische E-Mail-Vorlagen
Standardmäßig verwendet Ihre Anwendung beim Versenden von Dokumenten die globalen E-Mail-Vorlagen (konfiguriert in Ihren Unternehmenseinstellungen). Das Client-Management-System ermöglicht es Ihnen jedoch, dies für jeden einzelnen Client zu überschreiben.
- Navigieren Sie zur Unterregisterkarte **E-Mail-Vorlagen** im Profil eines bestimmten Kunden.
- Sie können eine völlig individuelle Betreffzeile und einen E-Mail-Text verfassen, die speziell auf sie zugeschnitten sind.
- Immer wenn Sie eine neue Rechnung oder einen neuen Kostenvoranschlag für diesen Kunden erstellen und auf „Per E-Mail senden“ klicken, ignoriert das System Ihre allgemeinen Einstellungen vollständig und verwendet ausschließlich deren personalisierte Vorlage. 
- Dadurch können Sie eine hochgradig personalisierte Note nahtlos und skalierbar beibehalten!

## 4. Das sichere, interaktive Kundenportal
Anstatt Ihren Kunden statische, langweilige PDF-Anhänge zu senden, die in ihrem Posteingang verloren gehen, umfasst Ihr Abonnement eine erstklassige, interaktive Webanwendung nur für Ihre Kunden.

### Passwortlose „Magic Link“-Authentifizierung
Ihre Kunden müssen kein Konto erstellen, sich kein anderes Passwort merken oder eine E-Mail bestätigen, um ihre Rechnungen einzusehen. 
- Wenn Sie eine Rechnung per E-Mail versenden, enthält diese einen robusten, kryptografisch sicheren „Magic Link“.
- Durch Klicken auf diesen Link werden sie sofort authentifiziert und direkt in ihrem privaten Kundenportal angemeldet.

### Die Anweisungsansicht und Massenaktionen
Sobald Ihr Kunde das Portal betritt, sieht er sich nicht nur eine einzelne Rechnung an; Sie betrachten ihre gesamte Beziehung zu Ihnen.
- **Zusammenfassung der Abrechnung:** Sie werden mit einer schönen Zusammenfassung begrüßt, in der der ausstehende Gesamtbetrag aufgeführt ist.
- **Online-Zahlungen:** Das Portal lässt sich direkt in Stripe integrieren. Ein Kunde kann eine ausstehende Rechnung einsehen und auf „Jetzt bezahlen“ klicken, um seinen Restbetrag sofort per Kreditkarte zu begleichen. Das System markiert Ihre interne Rechnung automatisch in der Millisekunde, in der die Transaktion abgeschlossen ist, als „Bezahlt“.
- **Massen-Download:** Für ihre eigenen Buchhaltungszwecke fordern Kunden häufig Kopien aller ihrer früheren Rechnungen an. Anstatt sie einzeln per E-Mail zu versenden, kann sich Ihr Kunde beim Portal anmelden, die Kontrollkästchen neben einem Dutzend früherer Rechnungen aktivieren und auf „Ausgewählte herunterladen“ klicken. Das System generiert eine übersichtliche ZIP-Datei, die alle PDFs gleichzeitig enthält.

### Integriertes Support-Ticketing
Hören Sie auf, Supportanfragen in unübersichtlichen E-Mail-Threads zu verlieren.
- Im Kundenportal kann Ihr Kunde auf „Neues Ticket öffnen“ klicken, um eine Frage zu stellen (z. B. „Können Sie meine Rechnungsadresse aktualisieren?“ oder „Ich habe eine Frage zu Position Nr. 3 auf meiner aktuellen Rechnung“).
- Dieses Ticket erscheint sofort in Ihrem Verwaltungs-Dashboard. 
- Sie können direkt von Ihrer Benutzeroberfläche aus antworten. Der Kunde erhält eine E-Mail-Benachrichtigung und kann direkt im Portal antworten. Dadurch wird sichergestellt, dass die gesamte Konversation im Thread bleibt und dauerhaft mit seinem CRM-Profil verknüpft ist.
