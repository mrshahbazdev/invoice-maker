# 客户管理和 CRM：深入指南

有效管理客户关系是成功运营服务或基于产品的业务的核心。将所有客户数据组织起来并在一个中心位置轻松访问，可确保您在与客户沟通、计费或支持时不会错过任何一个环节。 

**客户**模块充当您的整个客户关系管理 (CRM) 中心。从这个仪表板中，您可以查看与您互动的任何个人或企业的整个生命周期历史记录。

---

## 1. 创建和管理客户资料
每次互动都从客户资料开始。这是存储有关客户的所有关键信息的主记录。拥有完整的配置文件使您将来只需单击一下即可为其开具发票。

### 基本商业信息
创建新客户端时，您可以输入：
- **联系方式：** 他们的主要电子邮件地址和直线电话号码。此处提供的电子邮件严格由我们的自动邮件系统用于发送估价、发票和付款收据。
- **实际地址：** 您可以存储他们的完整帐单地址（街道、城市、州/省、邮政编码和国家/地区）。此信息会自动拉入您为他们生成的任何发票的“账单收件人”部分。

### 税务合规和法律领域
在许多司法管辖区，B2B（企业对企业）发票需要极高的法律准确性。
- **税号/增值税 ID：** 您可以将特定的纳税识别号 (TIN) 或增值税 ID 直接保存到客户资料中。每当您为该客户创建发票时，他们的税号都会自动嵌入到最终的 PDF 中，确保您的企业符合当地和国际税法，无需手动输入数据。

---

## 2. 交互式客户端仪表板
单击主列表中任何客户的姓名即可打开其专用的 360 度交互式仪表板。该屏幕为您提供有关您与他们的财务关系的即时、可操作的情报。

### 终生价值指标
在仪表板的最顶部，您将立即看到：
- **发票总额：** 发送给该客户的每张发票的总计。
- **总付费（终生价值）：** 您从他们那里获得的实际实现收入。
- **未结余额：** 他们目前在所有未结发票中欠您的确切金额。这可以帮助您在接受客户的新工作之前立即识别客户是否信誉良好。

### 文档历史记录
在财务指标下方，您可以立即访问他们的整个书面记录。 
- 您无需查找全球发票清单。该仪表板会过滤所有内容，仅显示特定于该客户的发票和估算。
- 您可以查看哪些发票是起草的、发送的、部分的或全额支付的。
- 您还可以查看发送给他们的所有估算，跟踪哪些估算被接受或拒绝。

## 3. 定制沟通渠道
每个企业的运作方式都不同。有些客户更喜欢正式、高度结构化的语言，而另一些客户则对轻松、友好的签到反应更好。每次向不同的人收取费用时，您不必重新编写电子邮件。

### 客户特定的电子邮件模板
默认情况下，您的应用程序在发送文档时使用全局电子邮件模板（在您的业务设置中配置）。但是，客户管理系统允许您根据每个客户覆盖此设置。
- 导航到特定客户配置文件内的 **电子邮件模板** 子选项卡。
- 您可以编写专门为他们量身定制的完全自定义的主题行和电子邮件正文。
- 每当您为该客户生成新的发票或估算并单击“通过电子邮件发送”时，系统将完全忽略您的常规设置并专门使用他们的个性化模板。 
- 这使您能够大规模地无缝地保持高度个性化的接触！

---

## 4. 安全的交互式客户端门户
您的订阅包括专为您的客户提供的高级交互式 Web 应用程序，而不是向您的客户发送在收件箱中丢失的静态、无聊的 PDF 附件。

### 无密码“Magic Link”身份验证
您的客户不需要创建帐户、记住另一个密码或验证电子邮件来查看他们的账单。 
- 当您通过电子邮件发送发票时，它包含一个强大的、加密安全的“Magic Link”。
- 单击此链接立即对他们进行身份验证并将他们直接登录到他们的私人客户门户。

### 语句视图和批量操作
一旦进入门户，您的客户不仅仅会查看一张发票；还会查看一张发票。他们正在审视与你的整个关系。
- **报表摘要：** 他们会收到一份精美的摘要，显示其未清偿总额。
- **在线支付：** 该门户直接与 Stripe 集成。客户可以查看未结发票并单击“立即付款”以使用信用卡立即结清余额。系统会在交易清算的毫秒内自动将您的内部发票标记为“已付款”。
- **批量下载：** 为了自己的会计目的，客户经常要求提供所有过去账单的副本。您的客户无需单独通过电子邮件发送它们，而是可以登录门户，选中十几张过去的发票旁边的复选框，然后单击“下载所选内容”。系统将生成一个简洁的 ZIP 文件，同时包含所有 PDF。

### 集成支持票务
不再在混乱的电子邮件线程中丢失支持请求。
- 在客户门户内，您的客户可以单击“打开新票据”来提出问题（例如，“您可以更新我的帐单地址吗？”或“我对最近账单上的第 3 行项目有疑问”）。
- 此票证立即出现在您的管理仪表板中。 
- 您可以直接从界面回复。客户将收到电子邮件通知，并可以直接回复门户，确保整个对话保持线索并永久链接到他们的 CRM 个人资料。
