# 费用跟踪和管理：详细指南

一个盈利的企业不仅仅在于你能带来多少收入，还在于你能带来多少收入。这是关于仔细监控有多少钱流出。 **费用**模块充当您的数字文件柜，确保您永远不会丢失收据，并且您始终确切地知道您的运营资金花在哪里。

---

## 1. 记录新费用
每当您购买软件、设备或出差时，都应该立即记录。导航到 **费用** 选项卡并单击 **记录费用**。

### 分类和组织
正确的分类是年底获得可操作的财务数据的关键。
- **费用类别：** 选择费用类型（例如“软件订阅”、“办公用品”、“旅行”、“餐饮和娱乐”）。这对流出进行分类，以便您的现金簿可以生成您的消费习惯的准确饼图。
- **金额和日期：** 记录支付的确切金额和交易发生的日期。
- **供应商/商家：** 准确记录您的付款人（例如“亚马逊”、“达美航空”、“微软”）。这可以帮助您确定一段时间内的经常性成本。

---

## 2. 收据和文件存储
如果您接受审核，电子表格中的行项目是不够的；您需要实际收据。该平台提供安全的、基于云的文档附件。

- **上传字段：** 记录费用时，您可以单击“附加文件”并上传 PDF 收据、餐厅账单的 JPEG 照片或数字交易的屏幕截图。
- **永久存储：** 该文件经过安全加密并永久附加到云中的费用记录。您实际上可以扔掉装满收据的物理鞋盒——您的整个审计跟踪都是数字化的并且可搜索。

---

## 3. 可计费费用（客户重新计费）
通常，您代表特定客户产生费用，他们根据合同有义务向您报销（例如服务器托管费、库存摄影购买或旅行门票）。

### 重新计费工作流程
该平台可以完美地处理这种复杂的会计操作：
1. 记录费用（例如，50 美元的库存照片）时，查找 **“分配给客户”** 下拉列表。
2. 选择为其购买的特定客户。
3. 选中标有 **“可计费”** 的框。
4. 系统在内部标记该费用。 
5. 下次您为该特定客户创建新发票时，系统将提醒您您有未开票的费用。只需单击一下，您就可以将 50 美元的库存照片成本作为行项目直接添加到发票中，确保您永远不会忘记获得自付费用的报销。

---

## 4. 全球金融一体化
费用模块并不存在于真空中。它深深地嵌入到**现金簿**和会计核心中。

- 您记录的每一项费用都会立即从主仪表板中的净利润指标中扣除。
- 如果您对费用分类错误，您可以随时对其进行编辑，现金簿数学将在整个平台上实时重新计算您的 P&L（损益）报表。
